# My personal website [![Actions Status](https://github.com/cfgnunes/cfgnunes.github.io/workflows/build/badge.svg)](https://github.com/cfgnunes/cfgnunes.github.io/actions)

Repository of my personal website published in <https://cfgnunes.github.io>.

## Getting Started

### Prerequisites

This section assumes Ubuntu 16.04 (also tested on Ubuntu 20.04), but the procedure is similar for other Linux distributions. The prerequisites is to install the following packages:

```sh
sudo apt -y install make bundler zlib1g-dev libxml2-dev libxslt1-dev
```

### Running the website

Use the following command:

```sh
make run
```

Now, just open it in your browser: <https://localhost:4000>

## Introduction

wxMP3val is a free front-end for the MP3val. Dedicated to MP3 files, wxMP3val lets you scan your music collection for corrupt files, incomplete downloads, and missing tags in an attempt to fix them.

![Screenshot](https://a.fsdn.com/con/app/proj/wxmp3val/screenshots/Main.png/max/max/1)

## Features

* Supports batch file (MP3) scanning and repairing;
* Once scanned, it shows details such as the audio layer version, meta-tag version, if the file was encoded using CBR (constant bit rate) and the condition.
